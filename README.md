# My Saree Store

This is the **exact clone** of Sudathi.com (downloaded with wget). UI matches 100%. Branding updated to "My Saree".

**To view:** Open `index.html` in a browser, or run a local server from this folder:

```bash
# From this folder:
python3 -m http.server 8080
# Then open http://localhost:8080
```

Asset paths are relative (`/cdn/...`), so use a local server for best results (some browsers restrict file:// for assets).

**Backup:** Your previous custom build is in `my-saree-store-old-backup/`.
